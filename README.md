# Star Wars Character App

This project is a web application that allows users to browse and search for Star Wars characters. It fetches data from the [SWAPI API](https://swapi.dev/) and presents it in a user-friendly interface.

## How to Run the Project

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd starwar
    ```

2.  **Install Dependencies:**
    Open your terminal and run the following command to install the necessary dependencies:
    ```bash
    npm install
    ```

3.  **Run the Development Server:**
    After the installation is complete, run this command to start the development server:
    ```bash
    npm run dev
    ```
    This will start the application, and you can view it in your browser at the address provided in the terminal (usually `http://localhost:5173`).

## Implemented Features

*   **Character Browser:** Displays a paginated list of Star Wars characters Cards.
*   **Search Functionality:** Allows users to search for characters by name.
*   **Character Details Modal:** Clicking on a character card opens a modal with detailed information, including name, height, mass, birth year, and homeworld, etc.
*   **Responsive Design:** The application is designed to work on different screen sizes.
